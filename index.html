Projects

Implementation of GPT w/BPE

I created a lightweight implementation of the pioneering transformer architecture that is used to power Large Language Models. Using PyTorch with Cuda support, I integrated the cutting edge technology behind these models, such as self-attention, tokenized word and position encoding, multi-layer perceptrons, and softmax to produce a model that emulated natural language. I then upgraded it with Byte-Pair Encoding, forced regex breaks, and special tokens to further minimize loss and save memory caused by the wastage of excess tokens.

Movie Reccommendation System (ML/Collaborative Filtering)

Using the MovieLens dataset, I worked with a teammate to build an machine learning algorithm that approximates how users would rate movies, based on how other users with similar tastes have rated them. Through estimating these ratings, we implemented a reccomendation system to reccommend a user new movies. This model can be generalized across any data stored in user-items fasion.

An easy to use trading platform for newbies that I created in a group project!

I collaborated with a couple developers to make a simple, easy-to-use application. This project aims to limit the complexities of traditional trading that many inexperienced beginners face when starting. Through construcing a backend and database architecture, I supported fundamental functionality, such as signin-up, managing a portfolio, and supervising a stock watchlist.

Spotify Track Popularity GUI

A fun and accessible data visualization & user interface project

Here I leveraged Spotify's Web API to create an interesting GUI capable of displaying dynamic data. Through following a simple pipelined process, users can enter a Spotify artist and receive a graph containing the artists most popular albums. Within each album, tracks are rated by popularity on Spotify's 0-100 scale.
